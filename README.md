## 🚀 What GuitarHelper Does

GuitarHelper is a desktop app for guitar practice. It brings key tools into one place so you can stay focused while you play.

Use it to:

- write down chords, riffs, and song parts
- tune your guitar with a real-time tuner
- view notes on a fretboard map
- keep time with a metronome
- move between practice tasks without opening other apps

It works on Windows and also supports Mac and Linux.

## 💻 Before You Start

You need:

- a Windows PC
- a mouse and keyboard
- a guitar
- speakers or headphones for the metronome
- a microphone if you want to use the tuner

For best results, close other audio apps before you tune or use the metronome.

## 🎼 Main Tools

### 🎸 Smart Notepad

Use the notepad to save song ideas, chord progressions, riff names, and practice goals.

Good uses:

- store chord charts
- note song sections like verse and chorus
- save tuning notes
- keep track of what you want to practice next

### 🎵 Real-Time Tuner

The tuner listens to your guitar through your mic and shows how close each string is to pitch.

Use it when:

- you tune before practice
- you switch to a different tuning
- you want to check one string fast

Tips for better tuning:

- use a quiet room
- place the mic close to the guitar
- pluck one string at a time
- let the string ring until the note settles

### 🧭 Fretboard Visualizer

The fretboard view helps you see where notes live on the neck. This can help with scales, chord shapes, and note finding.

Use it to:

- learn note names
- see pattern shapes
- connect music theory with the guitar neck
- build better fretboard memory

### 🥁 Metronome

The metronome helps you keep steady time. Set the tempo and play along.

Use it for:

- chord changes
- picking drills
- scale practice
- rhythm work
- slow practice before full speed

A steady click can help you keep your timing clean and even.

## 🪄 How to Use It for Practice

A simple practice flow:

1. Open the tuner and tune your guitar
2. Open the metronome and set a slow tempo
3. Use the notepad to write your practice plan
4. Open the fretboard view if you want to study notes
5. Practice one section at a time
6. Raise the tempo when you feel ready

This keeps your practice in one app and cuts down on switching between tools.

## 🛠️ Troubleshooting

### The app does not open

Try this:

1. Make sure the download finished
2. Open the file from the folder where you saved it
3. Right-click the app and choose Run as administrator
4. Restart your PC
5. Download the file again from the GitHub page

### The tuner does not detect sound

Try this:

1. Check that your microphone is on
2. Make sure the app has mic access in Windows settings
3. Move closer to the mic
4. Turn off other apps that use the microphone
5. Play one string at a time

### The metronome is too loud or too quiet

Try this:

1. Check your system volume
2. Check app sound settings if they are available
3. Use headphones if you want a clear click
4. Lower other audio sources

### The interface looks too small or too large

Try changing display scaling in Windows:

1. Open Settings
2. Go to System
3. Open Display
4. Change Scale until the text looks right
